
# 📊 Vrinda Store Sales Data Analysis (Excel Dashboard)

## 📌 Project Overview

This project analyzes **Vrinda Store's 2022 sales data** to identify key sales trends, customer behavior, and platform performance.
Using **Microsoft Excel**, the raw dataset was cleaned, analyzed, and transformed into an **interactive dashboard** to help businesses make data-driven decisions.

The dashboard provides insights into:

* Sales trends
* Customer demographics
* Top-performing states
* Order channels
* Order status distribution

---

## 🛠 Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Data Visualization
* Excel Slicers (Interactive Filters)

---

## 📂 Dataset

The dataset contains sales transactions from **Vrinda Store for the year 2022**.

Key columns include:

| Column       | Description                                     |
| ------------ | ----------------------------------------------- |
| Order ID     | Unique order identifier                         |
| Date         | Order date                                      |
| Gender       | Customer gender                                 |
| Age Group    | Customer age category                           |
| State        | Customer location                               |
| Channel      | Sales platform (Amazon, Flipkart, Myntra, etc.) |
| Category     | Product category                                |
| Amount       | Sales revenue                                   |
| Order Status | Delivered, Returned, Cancelled                  |

---

## 🧹 Data Cleaning Process

Before performing analysis, the dataset was cleaned using Excel.

Steps performed:

* Removed duplicate records
* Checked and handled missing values
* Standardized category names
* Converted date formats
* Verified order status values
* Organized columns for analysis

Excel Features Used:

* Remove Duplicates
* Filters
* Sorting
* Pivot Tables

---

## 📊 Dashboard Features

The Excel dashboard provides multiple visual insights:

### 1️⃣ Orders vs Sales Trend

Shows the relationship between **monthly sales revenue and number of orders**.

### 2️⃣ Top 5 States by Sales

Identifies the **states contributing the highest sales revenue**.

### 3️⃣ Orders by Age & Gender

Analyzes customer demographics and purchasing patterns.

### 4️⃣ Orders by Channel

Shows distribution of orders across platforms such as:

* Amazon
* Flipkart
* Myntra
* Ajio
* Meesho

### 5️⃣ Sales by Gender

Compares sales contribution between **men and women customers**.

### 6️⃣ Order Status Analysis

Displays the percentage of:

* Delivered orders
* Returned orders
* Cancelled orders
* Refunded orders

## 🔎 Key Insights

Important findings from the analysis:

* **Women contribute 64% of total sales**, making them the primary customer segment.
* **Amazon generates the highest number of orders (35%)**.
* **Maharashtra and Karnataka** are the top-performing states in terms of revenue.
* The majority of customers belong to the **adult age group**.
* **92% of orders are successfully delivered**, indicating efficient order fulfillment.

---

## 📈 Business Recommendations

Based on the analysis:

* Focus marketing campaigns on **female customers**
* Increase promotions on **Amazon and Myntra platforms**
* Expand logistics and marketing in **high-performing states**
* Target **adult customers** with personalized offers
